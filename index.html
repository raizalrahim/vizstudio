HOW TO USE

Getting Started

DROP / BROWSEUpload a music file (MP3, WAV, FLAC, OGG, M4A) into the upload zone to get started
STEMS ONLYSkip the upload zone — load stems directly into individual panels using each panel's STEM slot
▶ PLAYPlays all audio simultaneously — main track and all stems in sync
⏏ CHANGE TRACKSwap the main track without resetting your panel layout
✕ CLEAR TRACKRemove the main track — panels and stems stay intact
🎤 MICROPHONESwitch to live mic input — all panels visualize microphone audio in real time. Stops any playing audio.

Panel Controls

⠿ DRAGDrag the handle to reorder panels
⧉ DUPLICATECopy a panel with all its settings — stem not copied
🔇 MUTESilence a stem's audio without removing it
⚙ SETTINGSViz type, colors, sensitivity, label, background image
⛶ FULLSCREENExpand one panel to fill the entire screen
✕ REMOVERemove the panel and unload its stem

Visualization Types

BARSFrequency bar chart with gradient fill
WAVEGlowing time-domain waveform
CIRCLERadial shape that pulses with energy
MIRROR BARSBars mirrored above and below center
BLOBOrganic morphing shape that breathes and pulses with audio energy
OSCILLOSCOPERaw audio signal trace
SPECTRUM FILLFilled frequency area with gradient
RADIAL BARSBars radiating outward from center
RING BURSTConcentric rings that expand outward on energy peaks then fade — dramatic on bass-heavy music
LISSAJOUSGeometric looping curves drawn from audio signal — hypnotic and mathematical

Stems

+ LOAD STEMAssign a stem file to a panel — it visualizes only that stem's frequencies
🟢 LIVE DOTGreen pulsing = playing · Orange = paused · Dark = no stem
💡 Use LALAL.AI, Moises.app, or Audiostrip to split your song into stems — then load each into its own panel for accurate per-instrument visuals

Microphone

🎤 USE MICFound in ≡ menu — switches all panels to live microphone input. Any playing audio stops first.
⏹ STOP MICStops mic and returns to previous audio mode — file, stems, or empty
MIC LIVE BARRed pulsing bar appears while mic is active — stems are bypassed, all panels visualize mic input
💡 Mic audio is never played through your speakers — no feedback loop. Works best in a quiet environment or with headphones.

Presentation Mode

⛶ PRESENTHides all UI — panels fill the screen with pure visualizations
ESCExit presentation mode
💡 Start playing your music before entering presentation mode — controls are hidden inside it

Save & Load Layout

💾 SAVE LAYOUTExports panel config as JSON — band, viz type, colors, sensitivity, labels saved
📂 LOAD LAYOUTRebuilds panels from a saved JSON — audio files must be reloaded manually
💡 Audio files and background images are not saved — browsers block local file access between sessions for security reasons
